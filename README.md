# whistle.plugin-retry

Auto retry timeout requests for whistle.

## Usage

1. Install

```bash
npm install -g whistle.plugin-retry
```

2. Add rules

```
pattern plugin-retry://3000
```

you can replace `3000` with the timeout you want to set.
